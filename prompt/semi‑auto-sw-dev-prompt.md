# 指示1

最新のClaude Codeを使い、ほぼ全自動でSW開発するためのマニュアルをMCBSMDで出力せよ。
ほぼ全自動とはユーザーが関与する作業を以下のような重要な業務に絞ることをいう。

- SWの必要性やコンセプトを提示
- SWの開発に必要な重要な判断
- 最終的なSWの受入テスト
  以下のような作業はすべてClaude Codeが行う
- SW開発の計画立案(WBS/ガントチャート)
- 開発に必要な要員(AI)の管理
- 開発に必要なリソースの管理
- SW開発の進捗管理
- 要求仕様 (SRS) 作成
- セキュリティー設計
- SW仕様 (SWS) 作成
- SW実装
- モジュール単体テスト
- 結合テスト
- システムテスト (可能な限り)
- テスト消化曲線の監視
- バグカーブの監視
- ボトルネック/弱点領域へのリソース/人員(AI)追加
- 各成果物の段階的レビュー
- その他AIに可能なすべての業務
